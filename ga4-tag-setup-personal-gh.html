GA4 tag setup

This page loads the Google tag for measurement ID G-DPT471KXGG (Tempo Explore extension stream).

Open it once in a browser so the GA4 Web stream receives at least one hit. After that, the extension can send events via Measurement Protocol and they will be collected.

You can host this file on GitHub Pages, an internal site, or open it locally (file:// may not send data; use a local server or deploy somewhere).
